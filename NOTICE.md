# Attribution

PPTXMate is derived from [Office Agents](https://github.com/hewliyang/office-agents), originally created by Li Yang and contributors.

The repository retains the upstream Git history and internal `@office-agents/*` package names so authorship and compatibility remain visible. PPTXMate's PowerPoint-specific routing, bounded context, stable slide targeting, interruption recovery, clipboard image support, CC Switch integration, and macOS lifecycle tooling are maintained in this repository.
